SCPI Commands Documentation

This document provides the device SCPI (Standard Commands for Programmable Instruments) commands.

Basic Commands

  • *IDN?
    Queries the instrument's identity (manufacturer, model, serial number, and firmware version).
  • *RST
    Perform a hardware reset. WARNING: Connection will be clost.
  • *CLS
    Clears the status data structures.

Measurement Commands

  • MEAS:VOLT:DC?
    Measures the DC voltage.
  • MEAS:CURR:DC?
    Measures the DC current.
  • MEAS:RES?
    Measures the resistance.

Configuration Commands

  • CONF:VOLT:DC
    Configures the instrument to measure DC voltage.
  • CONF:CURR:DC
    Configures the instrument to measure DC current.
  • CONF:RES
    Configures the instrument to measure resistance.

Control Commands

  • OUTP ON
    Turns the output on.
  • OUTP OFF
    Turns the output off.
  • SYST:ERR?
    Queries the error queue.
